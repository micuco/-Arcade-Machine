# Pong - Arcade Machine

This project is an implementation of the Pong game for the Arcade Machine.

## Description

The project includes a spaceship editor and a Pong game. The player first builds a simple ship using a grid-based editor, then starts the game using that configuration.

## Features

### Editor
- Grid-based ship construction
- Drag and drop blocks
- Right click to remove blocks
- Validation of basic constraints before starting the game

### Pong
- Two paddles controlled by players
- Ball movement and collision
- Score system
- Ball reset after each point

## Controls

- Left player: W / S
- Right player: Up / Down arrows

## Technologies

- C++
- OpenGL

## How to run

Open the project in Visual Studio, build it, and run. Use the editor to create a ship, then press start to begin the game.

## Notes

Only the Pong version of the assignment was implemented.
